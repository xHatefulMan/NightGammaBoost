# NightGammaBoost

A lightweight C# WPF app that automatically boosts your screen gamma when the game scene gets too dark, making night scenes and dark environments much more visible.

> The application interface is in French.

## Features

### Automatic gamma boost
- Monitors the brightness of your screen in real time
- Automatically boosts gamma when brightness drops below a defined threshold
- Live **before/after preview** directly in the app

### Settings
- **Trigger threshold** — set the brightness level (%) below which the boost activates
- **Boost intensity** — set the gamma level applied when the threshold is reached
- **Global mode** — apply the boost to the entire screen permanently
- **Launch at Windows startup** in the background

### Multi-monitor support
- Select which screen to apply the boost to (Screen 1, Screen 2, etc.)

### Auto-activation by game
- Automatically enables the boost when a detected game is launched
- Supports: Minecraft, The Forest, Sons of The Forest, Valheim, and more

---

## Usage

1. Download `NightGammaBoost.exe` from the [Releases](https://github.com/xHatefulMan/NightGammaBoost/releases) page
2. Run it — no installation needed
3. Set your trigger threshold and boost intensity
4. Enable the boost for specific games or use global mode
5. The app runs in the background and boosts gamma automatically
